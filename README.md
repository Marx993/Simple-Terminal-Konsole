# Simple Terminal Konsole

A very simple terminal emulator/console application written in C++.

## Features

- Lightweight and simple terminal interface
- Written in pure C++
- Easy to compile and install

## Requirements

- C++17 compatible compiler (g++ or clang++)
- Linux/Unix system
- Standard C++ library

## Installation

### From Source

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Marx993/Simple-Terminal-Konsole.git
   cd Simple-Terminal-Konsole
   ```

2. **Compile:**
   ```bash
   make
   ```

3. **Install:**
   ```bash
   sudo make install
   ```
   This will install the binary to `/usr/local/bin/konsole`

### Usage

After installation, run:
```bash
konsole
```

Or run directly from the build directory:
```bash
./konsole
```

## Uninstallation

To remove the installed binary:
```bash
sudo make uninstall
```

## Development

### Build

### Clean
